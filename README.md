# Burger Builder Project 
Burger Builder Project is an application which creating a buger.



## Installation
Use the package manager npm to install project and to run.
```bash
cd app
npm install 
npm start
```
## Build With

 * React
 * React-Router
 * Redux
 * CSS-Modules
 * Firebase
 * Axios
